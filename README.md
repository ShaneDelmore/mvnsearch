# mvnsearch

Command line utility for searching Java/Scala library

## Install

via [conscript](https://github.com/n8han/conscript/)

```
 $ cs tototoshi/mvnsearch
```

## Usage

```
Usage: mvnsearch [options] <word>

  --rows <value>

  <word>
```
